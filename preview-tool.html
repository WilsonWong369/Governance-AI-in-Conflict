📄 QMD Preview Tool

📋 Quick Access

Use these buttons to quickly preview your QMD files:

  • Preview PDF: Convert Abstract.qmd to PDF and open it
  • Preview HTML: Convert Abstract.qmd to HTML and open it
  • VS Code Task: Use Ctrl/Cmd + Shift + P → "Tasks: Run Task" → "Preview QMD as PDF"

📁 Generated Files

Your preview files will be created in the Abstract folder:

Abstract_academic.pdf Abstract.html Abstract.qmd

📄 Academic PDF Format

The academic PDF is formatted to match standard research paper requirements:

  • Title Page: Centered title, author, course, professor, and date (no page number)
  • Table of Contents: Automatic TOC with page references (Roman numerals)
  • Page Numbers: Bottom center, Arabic numbers for content (1, 2, 3...)
  • Font: Times New Roman, 12pt throughout
  • Spacing: Double-spaced text (24pt leading)
  • Margins: 1.25 inches on all sides
  • Paragraphs: First-line indent (0.5 inches)
  • Alignment: Justified text
  • Headings: Bold, left-aligned, 12pt

📋 Document Structure

The PDF automatically generates the following structure:

  1. Title Page - No page number
  2. Table of Contents - Roman numerals (ii, iii, etc.)
  3. Content Pages - Arabic numbers (1, 2, 3, etc.)

The table of contents is automatically generated from your heading sections (lines starting with #).

📝 Title Page Fields

Add these fields to your QMD frontmatter for a complete title page:

  • title: - Paper title
  • author: - Your name
  • course: - Course name (optional)
  • professor: - Professor name (optional)
  • date: - Submission date (optional, defaults to today)

💡 Tips

  • Edit your content in Abstract/Abstract.qmd
  • Use the preview buttons to see your changes
  • PDF files will open with the vscode-pdf extension
  • HTML files will open in your default browser
